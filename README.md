# Faria Lima Backstage

Repositório do site farialimabackstage.com.br

## Estrutura

```
.
├── index.html          # Homepage
├── style.css           # Estilos
├── script.js           # Interações
├── favicon.svg         # Favicon
├── robots.txt          # Configuração para crawlers
├── sitemap.xml         # Mapa do site para Google
└── netlify.toml        # Configuração de hosting (Netlify)
```

## Deploy

Site hospedado na Netlify com deploy automático a partir deste repositório.
Toda alteração na branch `main` é publicada automaticamente em farialimabackstage.com.br.

## Como atualizar conteúdo

1. Edite os arquivos diretamente pelo GitHub (botão "Edit" - ícone de lápis)
2. Faça commit
3. Aguarde ~30 segundos
4. Site atualizado ✓
